# Royal Jubilant Design System

## Color Palette (Single Source of Truth)

### Brand Colors
| Role | Hex | Usage |
|---|---|---|
| Primary Gold | `#C9A961` | Primary CTAs, active states, gold accents |
| Secondary Gold | `#A68A3F` | Hover states for gold elements |
| Light Gold | `#D4B875` | Subtle highlights, borders |
| Primary Navy | `#0A1F44` | Headings, primary buttons (navy), text |
| Navy Hover | `#0A1F44` at 90% opacity | Hover for navy buttons |

### Neutral Colors
| Role | Hex | Usage |
|---|---|---|
| Background | `#FFFFFF` | Page background |
| Surface | `#F9FAFB` | Card backgrounds, sections |
| Border | `#E5E7EB` | Borders, dividers |
| Body Text | `#6B7280` | Body copy, descriptions |
| Muted Text | `#9CA3AF` | Placeholders, captions |
| Heading Text | `#0A1F44` | All headings |
| White | `#FFFFFF` | Text on dark backgrounds |

### Status Colors
| Role | Hex | Usage |
|---|---|---|
| Success | `#28A745` | Online status, success messages |
| Error | `#DC3545` | Error states |
| Warning | `#FFC107` | Warning states |

## Border Radius
| Size | Value | Usage |
|---|---|---|
| Small | 8px | Inputs, small buttons |
| Medium | 12px | Cards, buttons |
| Large | 20px | Hero elements, modals |

## Shadows
| Level | Value |
|---|---|
| Small | `shadow-sm` |
| Medium | `shadow-md` |
| Large | `shadow-2xl` |

## Button Styles
| Type | Background | Text | Hover |
|---|---|---|---|
| Gold Primary | `bg-[#C9A961]` | white | `hover:bg-[#A68A3F]` |
| Navy Primary | `bg-[#0A1F44]` | white | `hover:bg-[#0A1F44]/90` |
| White | `bg-white` | `#0A1F44` | `hover:bg-gray-100` |
| Outline | `border border-[#C9A961]` | `#C9A961` | `hover:bg-[#C9A961]/10` |
